# 1min-stock-data

東証銘柄の1分足データを取得するCLIツールです。

## 機能

- 指定した東証銘柄の1分足データを取得
- 取得期間を指定可能（1日、1週間、1ヶ月、1年など）
- 取引時間中のデータのみを表示（休み時間のデータは自動除外）
- 始値、高値、安値、終値、出来高を表示
- 日本時間での時刻表示
- テーブル形式またはJSONL形式での出力

## インストール

```bash
# 依存関係をインストール
pnpm install
```

## 使用方法

```bash
# 基本的な使用方法（今日の1分足データ）
pnpm start <銘柄コード>

# 例：トヨタ自動車（7203）
pnpm start 7203

# 期間を指定してデータ取得
pnpm start 7203 --period 1w    # 1週間
pnpm start 7203 --period 1m    # 1ヶ月
pnpm start 7203 --period 6m    # 6ヶ月

# JSONL形式で出力
pnpm start 7203 --jsonl
pnpm start 7203 --period 2d --jsonl

# 例：ソフトバンクグループ（9984）
pnpm start 9984 --period 1w

# または直接tsxで実行
npx tsx src/index.ts 7203
npx tsx src/index.ts 7203 --period 1m --jsonl
```

### 期間指定

以下の形式で過去のデータ取得期間を指定できます：

- **日数**: `1d`, `2d`, `7d` など
- **週数**: `1w`, `2w`, `4w` など  
- **月数**: `1m`, `2m`, `6m`, `12m` など
- **年数**: `1y`, `2y` など

デフォルトは `1d`（今日）です。

### 出力形式

- **テーブル形式（デフォルト）**: 見やすい表形式で出力
- **JSONL形式（`--jsonl`）**: 各行がJSONオブジェクトのJSONL形式で出力


### JSONL形式

```jsonl
{"symbol":"7203.T","timestamp":"2025-06-16T00:00:00.000Z","date":"2025-06-16","time":"09:00","open":2580.5,"high":2581.5,"low":2570,"close":2572.5,"volume":0}
{"symbol":"7203.T","timestamp":"2025-06-16T00:01:00.000Z","date":"2025-06-16","time":"09:01","open":2573,"high":2577,"low":2569,"close":2570.5,"volume":243200}
{"symbol":"7203.T","timestamp":"2025-06-16T00:02:00.000Z","date":"2025-06-16","time":"09:02","open":2570.5,"high":2577.5,"low":2570.5,"close":2575,"volume":224700}
...
```

#### JSONLフィールド説明

- `symbol`: 銘柄コード（Yahoo Finance形式）
- `timestamp`: ISO 8601形式のタイムスタンプ（UTC）
- `date`: 日付（YYYY-MM-DD）
- `time`: 時刻（日本時間、HH:MM）
- `open`, `high`, `low`, `close`: 株価（円）
- `volume`: 出来高

## 対応銘柄

- 東証に上場している全銘柄
- 銘柄コードのみを指定（自動的に`.T`サフィックスが追加されます）

## 技術仕様

- **言語**: TypeScript
- **実行環境**: Node.js (tsx使用)
- **データソース**: Yahoo Finance API (yahoo-finance2)
- **パッケージマネージャー**: pnpm

## 注意事項

- 取引時間外はデータが取得できない場合があります
- Yahoo Finance APIの制限により、大量のリクエストは避けてください
- データの正確性については、投資判断に使用する前に他のソースでも確認してください
