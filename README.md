# 1min-stock-data

東証銘柄の1分足データを取得するCLIツールです。

## 機能

- 指定した東証銘柄の今日の1分足データを取得
- 取引時間中のデータのみを表示（休み時間のデータは自動除外）
- 始値、高値、安値、終値、出来高を表示
- 日本時間での時刻表示

## インストール

```bash
# 依存関係をインストール
pnpm install
```

## 使用方法

```bash
# 基本的な使用方法
pnpm start <銘柄コード>

# 例：トヨタ自動車（7203）
pnpm start 7203

# 例：ソフトバンクグループ（9984）
pnpm start 9984

# または直接tsxで実行
npx tsx src/index.ts 7203
```

## 対応銘柄

- 東証に上場している全銘柄
- 銘柄コードのみを指定（自動的に`.T`サフィックスが追加されます）

## 技術仕様

- **言語**: TypeScript
- **実行環境**: Node.js (tsx使用)
- **データソース**: Yahoo Finance API (yahoo-finance2)
- **パッケージマネージャー**: pnpm

## 注意事項

- 取引時間外はデータが取得できない場合があります
- Yahoo Finance APIの制限により、大量のリクエストは避けてください
- データの正確性については、投資判断に使用する前に他のソースでも確認してください
